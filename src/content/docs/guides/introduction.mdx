## Overview
ProcessSpy is a powerful tool designed to monitor and analyze system processes in real-time. It provides users with detailed insights into process behavior, resource usage, and performance metrics.

The tool uses native APIs to gather information about running processes, ensuring accurate and up-to-date data. With its user-friendly interface and advanced filtering options, ProcessSpy makes it easy to identify resource-intensive processes, troubleshoot performance issues, and optimize system performance.

## Installation


<Steps>

1. Download the latest version of ProcessSpy (.dmg file) from the [official website](https://process-spy.app).

2. Open (mount) the .dmg file

3. Drag and drop the ProcessSpy application into your Applications folder

</Steps>

## License
Default version of ProcessSpy is free to use. Licensed version unlocks additional features such as more detailed process information, history recording etc. For more information see [offical page](https://process-spy.app).

License key can be purchased on [Gumroad](https://rob3rth.gumroad.com/l/yzukse) and activated in the application by navigating to the "ProcessSpy" menu and selecting `Enter license key...`.

## Updates
ProcessSpy includes a built-in update mechanism that checks for new versions automatically. When an update is available, you will be prompted to download and install it. You can also manually check for updates by navigating to the "ProcessSpy" menu and selecting `Check for Updates`.
